校园二手书交易平台

这是一个面向高校师生的闲置书籍在线交易系统，帮助学生将手中闲置的专业教材、课外书籍通过平台转让给需要的同学，实现书籍价值的循环利用。

卖书的学生毕业后书籍闲置扔了可惜，通过平台可以一键发布快速出售。买书的学生买新书太贵，学长学姐的书便宜又实用还支持校内交易。平台集中管理有审核把控，二手书流通信息不再散乱。

后端采用Spring Boot 3.2搭配MyBatis-Plus实现CRUD零SQL，使用Sa-Token轻量级Token认证，MySQL 8管理11张表完整数据模型，Redis做会话缓存。
前端采用Vue3加TypeScript开发H5移动端和PC管理端，移动端使用Vant4组件库有良好的移动端体验，工程化用Vite 5快速构建。

用户端功能包括首页浏览搜索、分类筛选、书籍详情收藏、发布闲置书籍、购物车结算、完整订单流程从下单到支付发货收货、以及即时消息咨询和个人中心。
管理端功能包括仪表盘统计、书籍审核、订单管理、用户管理、分类管理、公告管理和操作日志审计。

项目亮点是具备完整业务闭环从发布到审核展示购买支付发货收货评价，前后端分离API优先支持多端接入，校园场景适配支持书籍成色评级和分类管理，AOP日志实现管理员操作全程可追溯，响应式设计H5端适配各种手机屏幕。

适用于毕业季书籍转让、专业教材循环使用、校园社团书籍共享、课程参考书交易等场景。
